C

Uno de los primeros proyectos que desarrollé fue , César, un programa diseñado para encriptar y desencriptar mensajes utilizando el método empleado por el célebre personaje romano. Este programa recibe como argumentos la clave y la frase a encriptar.

Otro proyecto sencillo que realicé fue Scrabble, una simulación para dos jugadores del clásico juego de mesa homónimo.

También desarrollé un sistema de herencia de sangre, donde, utilizando recursión, se realiza un seguimiento para predecir el tipo de sangre probable en la descendencia.

Si se tiene una imagen pixelada y se desea recuperar la calidad, creé un programa para recuperarla. Además, desarrollé un programa para aplicar filtros a las imágenes, Filtro, donde se puede pasar la imagen y el filtro deseado como argumentos. En este proyecto, también adquirí conocimientos sobre gestión de memoria, incluyendo como asignarla y liberarla eficientemente.

Python

La mayoria de programas que he desarrollado en Python son recreaciones de aquellos que inicialmente realicé en C, utilizados como práctica adicional. También he trabajado en otros proyectos que emplean HTML y el framework Flask.

Por ejemplo, Finance es un proyecto en el que, empleando Flask y HTML, diseñé una página de compra de stocks, un portfolio por así llamarlo.

Un ejercicio que me resultó particularmente interesante fue Lectura , el cual analiza un texto para determinar el nivel educativo del autor.

SQL

Los proyectos desarrollados en SQL consisten en realizar diversas consultas en diferentes bases de datos.

Por ejemplo, en el proyecto Movies, realizo múltiples consultas relacionadas con nombres, fechas, directores, años, estrellas y ratings en una base de datos de películas.

Otro proyecto es Songs, donde trabajo con una base de datos que contiene el top 100 de canciones de Spotify en 2018. Aquí, realizo consultas basadas en nombres, artistas, años y características del estilo musical.

Además, he realizado trabajos en SQL para la carrera, como los proyectos Automóviles y Hoteles. En estos proyectos, además de llevar a cabo consultas, implementé funciones y vistas según los requerimientos específicos.

Java

En este caso, este lenguaje ha sido el que más he utilizado a lo largo de la carrera, especialmente en Programación Orientada a Objetos (POO) y Estructuras de Programación. He desarrollado una aplicación para la gestión de un hospital y otra para una empresa de transportes, ambas con un enfoque orientado a objetos. Pero además, he creado un reproductor de música, centrado en el uso de estructuras de datos como listas, colas, pilas y árboles, que, a mi parecer, es el proyecto más interesante de todos los que he llevado acabo en Java.
